我們來解這兩題關於**屬性集閉包（X⁺）**與**函數依賴推導**的問題：

---

## ✅ 第 1 題：驗證 `X → CWYZ`

已知：
- F = { X → YZ, Z → CW }

---

### 🔎 Step-by-step 計算 `X⁺`：

1. 初始：`X⁺ = {X}`  
2. 根據 X → YZ：`X⁺ = {X, Y, Z}`  
3. 接著，Z ∈ X⁺，套用 Z → CW：`X⁺ = {X, Y, Z, C, W}`  

---

### ✅ 結果：

X⁺ = {X, Y, Z, C, W}  
⇒ 所以可推出：`X → CWYZ ✅` → **成立**

---

## ✅ 第 2 題：求 `X_F⁺` （X 的閉包）

已知：
- F = { X → YZ, Z → CW, W → V, S → T }

---

### 🔎 Step-by-step 計算 `X⁺`：

1. 初始：`X⁺ = {X}`  
2. 用 X → YZ：`X⁺ = {X, Y, Z}`  
3. Z ∈ X⁺，Z → CW：`X⁺ = {X, Y, Z, C, W}`  
4. W ∈ X⁺，W → V：`X⁺ = {X, Y, Z, C, W, V}`

X⁺ = **{X, Y, Z, C, W, V}**
